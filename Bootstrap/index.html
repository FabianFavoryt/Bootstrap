HTML Element

Strukturerar, ett eller flera attribut innehållandes ett namn och ett värde.
De flesta HTML element består av en start respektive sluttagg och kan däremellan bestå av olika innehåll.

HTML (generellt)

(HyperText Markup Language) En av webbens grundläggande byggsten.
HTML är ett märkspråk. HTML har endast i uppgift att ge struktur

ID

ID måste vara unik inom html dokument.
ID används för att specifiera en specifik stilmall.

Class

Class används för att ange en klass för ett HTML-Element.
Flera HTML-Element kan dela samma klass

Meta (ordet)

Meta är ett ord som kommer från det grekisa språket.
Ordet betyder mellan bakom eller efter

Responsive

Ordet används oftast för att benämna att en hemsida är mobilanpassad.
För att kunna få till en mobil anpassad sida använder vi bootstrap.

Http

HTTP står för Hyper Text Transfer Protocol.
Kommunikation melan datorer och webbservar sker genom att skicka HTTP Requests och ta emot HTTP Responses.

Https

HTTPS står för Hypertext Transfer Protocol Secure.
Den ser till så att besökare har en säker uppkoppling till din hemsida.

DNS

DNS står för Domain Name System.
DNS håller en lista som är kopplad med alla resurser så som IP-Adresser.

CDN

CDN står för Content Delivery Network.
CDN hänvisar till en grupp av servrar som arbetar för att tillhandahålla snabb leverans av internetinnehåll.

Compressed data

Compressed data misnkar storleken på filer för att förbättra laddningstider och begränsa bandbreddsanvändning.

Local bootstrap

Local bootstrap är att man har laddat ner en fil.
Man har alltså ett bootstrap biblotek på datorn.

JQuery

JQuery ändrar allt innehåll i alla p element.

Flexbox

Flexbox är en endimensionell layoutmetod.
Den används för att ordna föremål i rader eller kolumner.

Grid

Grid är ett rutnätbaserat layoutsystem.
Det används för att göra det lättare att designa webbsidor utan att behöva använda flytningar och positionering.

Code snippets

Code snippets är små block kod som är återanvändbara.
Det kan infoga hela klasser eller metoder men även vanliga kodblock.

WASM

WASM står för WebAssembly.
WASM är designat som ett bärbart kompileringsmål vilket möjliggör distribution på webben för en klient och serverapplikationer.

Chromium

Chromium är en webbläsare.
Chromium är skapad från öppen källkod, chromium har nästan exakt samma funktioner som chrome. Gränssnittet är utvecklat för att vara minimaliskt, snabbt och mer intuitivt.

SVG

SVG står för Scalable Vector Graphics.
SVG används för att definiera grafik för webben.

png24

png24 är stöder 24-bitars färg.
png24 bevarar de subtila variationerna i ljusstyrka och nyans som finns i fotografier.
